AI Scripting Help

Intro

In this tutorial, you will learn how to create your own AI script. This script is what will control your avatar throughout the game. The more complex you make it, the more interesting your avatar will become. By scrolling through this page, you will find all the necessary information to correctly write code for your avatar. At any time, you can press on the sidebar buttons to jump forward to any section of your choice. Once the tutorial is finished, close the tab and return to the Create Actor window to implement everything you've just learned.

How to Begin

For your AI script to function properly, you must write it in such a way that our program can understand it. Think of our program as a dog. Unfortunately, it only knows certain commands. If you ask it to do something that is not in its vocabulary, it will not understand and return an error. These commands must also be written in such a way so that they make sense syntatically.

Action Words

The following words are actions that the actor can perform. They can be utilized in a variety of ways to help your avatar surive in the world. Each action will have an accompanying example in the scripting language to help better understand how it is used.

Attack

Attack in the direction that your avatar is facing. Attacking another avatar will cause that avatar to lose health.

Drop

Drop (if possible) whatever the actor is holding, in the direction that the actor is currently facing

Direction

Returns a list of directions to a specific coordinate. Accepts a tuple. Example calls: direction(nearest(WATER)), or direction (0 , 5). The first example will give you a list of directions to the nearest water element. The second will give you a list of directions to location (0,5) on the map.

Eat

If you are currently holding food, actor will eat, and thus regenerate health.

Flee

Action performs the opposite of a list of directions to something.

Harvest

Attempt to harvest in the direction that you are facing.

Nearest

Returns the coordinate of the nearest cell with the specified attribute.

Pickup

Attempt to pickup an object in the direction that you are facing. If no object is present nothing happens

Scavenge

Search for the nearest food source.

Sleep

Puts your avatar to sleep, for a specified amount of time

Walk

Takes in a list of directions as an argument and performs them. The list can also be only one direction.

Actor Information

The following words are available to obtain characteristics of the current actor. They are useful to use as the 'conditionals' in your script. For example: 'if myhealth > 50'

mydirection

Gives the direction that the current actor is facing.

myfood

Returns 'True' if the current actor is holding food. Otherwise will return 'False'.

myhealth

Returns the current health of the actor

mylocation

Returns the current location of the actor in the form of a 2-tuple of integers

asleep

Returns 'True' if the actor is asleep, otherwise will return 'False'

myrock

Returns 'True' if the actor is holding a rock, otherwise will return 'False'.
